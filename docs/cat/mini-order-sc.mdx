## ¿Qué es?

El **Mini Order** es el sistema de códigos de seguimiento de SYNTIcat. Cada pedido que un cliente envía desde el carrito recibe automáticamente un código único en formato **SC-XXXX** (ejemplo: SC-0001, SC-0042, SC-0123). Este código viaja en el mensaje de WhatsApp y queda guardado como archivo JSON en tu espacio del servidor.

## ¿Dónde lo encuentro?

**En la landing (lo que ve tu cliente):**
- El código SC-XXXX aparece en el mensaje de WhatsApp al finalizar el pedido
- También aparece en la pantalla de confirmación después de enviar el pedido

**En el servidor:**
- Los pedidos se guardan en `storage/tenants/{tu_tenant_id}/orders/2026/03/SC-XXXX.json`
- Organizados por año y mes automáticamente

## ¿Qué hace?

- **Genera un código secuencial único** para cada pedido (SC-0001, SC-0002, SC-0003...)
- **Guarda el pedido completo como JSON**: productos, cantidades, variantes, precios, total y fecha
- **Incluye el código en el mensaje de WhatsApp** para que ambos (tú y el cliente) tengan la misma referencia
- **Pantalla de confirmación**: El cliente ve el código SC-XXXX después de enviar su pedido

### Formato del JSON guardado

Cada pedido se guarda con:
- Código SC-XXXX
- Lista de productos con nombre, variante, cantidad y precio
- Total del pedido
- Fecha y hora
- ID del tenant

### Para qué sirve el código

| Uso | Ejemplo |
|-----|---------|
| **Identificar pedidos** | "¿Tu pedido es el SC-0047? Déjame revisarlo" |
| **Evitar confusiones** | Si tienes 3 pedidos similares, el código los distingue |
| **Profesionalismo** | Le da imagen de negocio organizado |
| **Seguimiento** | "Tu pedido SC-0047 ya está listo para recoger" |

## ¿Qué NO hace?

- **No rastrea estados** (preparando, enviado, entregado) — eso se coordina por WhatsApp
- **No envía notificaciones** automáticas al cliente sobre su pedido
- **No tiene panel de órdenes** visible en el dashboard (próximamente)
- **No genera factura** a partir del pedido
- **No permite cancelar** un pedido desde la plataforma
- **No tiene historial de pedidos** visible para el cliente

</Note>

## Ejemplo práctico

### Caso: Tienda de zapatos "PasoFirme" en San Cristóbal

Andrea tiene una tienda de zapatos con plan Anual. Un cliente pide:
- 1× Bota montañera (Talla 42) — $35
- 1× Par de medias — $5

Al tocar "Finalizar pedido", se genera el código **SC-0023**. El mensaje que recibe Andrea por WhatsApp:

```
🛍️ Nuevo pedido SYNTIcat
📦 Orden: SC-0023

• 1× Bota montañera (Talla 42) — REF 35.00
• 1× Par de medias — REF 5.00

💰 Total: REF 40.00

Código de seguimiento: SC-0023
```

Andrea confirma: *"Hola, tu pedido SC-0023 está listo. ¿Lo recoges o te lo envío? Pago Móvil Banesco al 04141234567"*

El archivo `SC-0023.json` queda guardado en el servidor para referencia futura.

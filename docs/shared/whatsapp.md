WhatsApp es el canal principal de SYNTIweb. Todos los pedidos, consultas y contactos de tus clientes llegan directo a tu WhatsApp.

## Agregar tu número de WhatsApp

1. Entra a tu panel de administración
2. Ve a la pestaña **Tu Información**
3. En el campo **WhatsApp ventas** ingresa tu número con código de país

**Formato correcto:** `584121234567` (58 + código de área + número)

<Warning>
  Escribe el número completo con el código de país (58 para Venezuela). Sin el código, los clientes no podrán escribirte desde fuera del país.
</Warning>

## Dos líneas de WhatsApp (plan Crecimiento+)

Si tienes un número para ventas y otro para soporte, puedes configurar ambos. El cliente elige a cuál escribir.

- **WhatsApp ventas** → para pedidos y consultas generales
- **WhatsApp soporte** → para preguntas o reclamos

## Cómo llega el pedido

Cuando un cliente hace un pedido desde tu página, SYNTIweb construye un mensaje automático con todos los detalles y abre WhatsApp listo para enviártelo. Tú recibes algo así:

```
🍽️ Pedido desde Tu Negocio

• 2× Pizza Margarita — REF 8.00 c/u
• 1× Refresco — REF 2.00

💰 Total: REF 18.00
```

## Ejemplo práctico

### Caso: Barbería "El Corte" en Maracaibo

Pedro configura su WhatsApp ventas: `584141234567`. Cuando un cliente ve el servicio "Corte clásico" en la página, toca **"Escríbenos"** y se abre WhatsApp con el número de Pedro automáticamente. Pedro confirma la cita sin salir de WhatsApp.

<Tip>
  Activa las notificaciones de WhatsApp para no perderte ningún pedido. Si tienes empleados, el segundo WhatsApp (plan Crecimiento+) te permite separar ventas de soporte.
</Tip>

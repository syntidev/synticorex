SYNTIweb tiene planes para cada tamaño de negocio. Todos incluyen tu página publicada, QR descargable y administración desde el celular.

## SYNTIstudio

Para cualquier negocio que necesite página web completa.

| | Oportunidad | Crecimiento | Visión |
|---|---|---|---|
| **Precio** | REF 99/año | REF 149/año | REF 199/año |
| **Productos** | 20 | 50 | Ilimitados |
| **Servicios** | 3 | 6 | 9 |
| **Fotos por producto** | 1 | 1 | 3 fotos (slider) |
| **Secciones** | Básicas | + Acerca de, Testimonios | + FAQ + Sucursales |
| **Paletas de color** | 10 | 17 | 17 + paleta propia |

## SYNTIfood

Para restaurantes, areperas, pastelerías y negocios de comida.

| | Básico | Semestral | Anual ⭐ |
|---|---|---|---|
| **Precio** | REF 9/mes | REF 39/semestre | REF 69/año |
| **Ítems en menú** | 50 | 100 | 150 |
| **Fotos de categoría** | 6 | 12 | 18 |
| **Tasa BCV automática** | ✗ | ✅ | ✅ |
| **Extras y opciones** | ✗ | ✅ | ✅ |
| **Pedido Rápido → WhatsApp** | ✗ | ✗ | ✅ |

## SYNTIcat

Para tiendas de ropa, proveedores y negocios con catálogo de productos.

| | Básico | Semestral | Anual ⭐ |
|---|---|---|---|
| **Precio** | REF 9/mes | REF 39/semestre | REF 69/año |
| **Productos** | 20 | 100 | Ilimitados |
| **Imágenes por producto** | 1 | 3 | 6 |
| **Carrito de compras** | ✗ | ✅ | ✅ |
| **Código de orden SC-XXXX** | ✗ | ✗ | ✅ |
| **Variantes (talla/color)** | Solo simple | Talla + color | Todas + opciones libres |

<Note>
  Los precios están en REF (referencia dólar). El equivalente en bolívares se calcula automáticamente con la tasa BCV del día. Puedes configurar cómo se muestran los precios desde **Tu panel → Configuración → Moneda**.
</Note>

SYNTIWEB — Documento Fundacional de Marca, Arquitectura y Estrategia SaaS
Versión: 1.0
Fecha: Febrero 2026
Propósito: Definir la estructura estratégica de marca, naming, arquitectura tecnológica y lineamientos de crecimiento para la plataforma SYNTI.
________________________________________
1. Visión Estratégica del Proyecto
SYNTI nace como una infraestructura de presencia digital autónoma, diseñada para permitir que negocios pequeños y medianos tengan sitios web funcionales, editables y escalables sin dependencia técnica.
El enfoque central del producto no es “crear páginas web”, sino proporcionar:
•	Infraestructura web automatizada
•	Motor multi-tenant escalable
•	Plataforma SaaS de despliegue instantáneo
•	Ecosistema white-label listo para expansión
________________________________________
2. Arquitectura de Marca (Brand Stack)
La estructura de marca debe permitir crecimiento sin rebranding en el futuro.
Marca madre
SYNTIdev
Representa:
•	la ingeniería
•	el laboratorio de desarrollo
•	la infraestructura tecnológica
•	los frameworks internos
•	la investigación y productos futuros


Producto comercial principal
SYNTIWeb™
Producto orientado al mercado masivo: creación y administración autónoma de sitios web.
Motor tecnológico interno
SYNTI Engine™ / SYNTI Core™
Nombre interno del framework y sistema que impulsa la plataforma.
Puede mantenerse invisible al cliente final.
Panel administrativo
SYNTI Console™
Interfaz de control del sistema SaaS y panel de clientes.

4. Modelo Multi-Tenant Estratégico
La plataforma debe operar bajo arquitectura multi-tenant desde el inicio.
Principios:
•	Una sola aplicación
•	Múltiples clientes
•	Datos aislados por tenant
•	Deploy único afecta a todos
•	Dominios múltiples apuntando al mismo core
Ejemplo de clientes:
negocio.menuaqui.com
negocio.webzona.net
negocio.micarta.vip
Todos resuelven al mismo sistema central.
Beneficios:
•	Escalabilidad inmediata
•	Costos marginales mínimos
•	Mantenimiento centralizado
•	Capacidad SaaS real

6. Estrategia de Posicionamiento
Diferenciación clave del producto:
No vender “páginas web”.
Vender infraestructura web automática.
Narrativa estratégica:
•	Web autónoma
•	Cero mantenimiento técnico
•	Despliegue instantáneo
•	Edición directa por el negocio
•	Plataforma preparada para escalar

8. Principios de Diseño del Producto
Reglas estructurales:
1.	El usuario no diseña; configura.
2.	El sistema define límites seguros de edición.
3.	La infraestructura debe escalar sin reinstalaciones.
4.	Las actualizaciones deben ser centralizadas.
5.	El dominio es parte del producto.
6.	El panel debe ser simple, directo y en contexto visual.

9. Estrategia Comercial Inicial
Fase inicial:
•	Enfoque en pequeños negocios
•	Dominio incluido
•	Panel simple editable
•	Pago inicial accesible
•	Modelo híbrido: setup + mantenimiento opcional
Fase crecimiento:
•	automatización de creación de clientes
•	subdominios masivos
•	dashboard master
•	white-label para agencias
Fase SaaS:
•	billing automático
•	provisioning automático
•	escalabilidad cloud
•	marketplace de plantillas

10. Principio Estratégico Central del Proyecto
SYNTI no es un constructor web.
Es una infraestructura distribuida de presencia digital donde:
•	el dominio es el punto de entrada
•	el engine es el sistema invisible
•	la plataforma escala como SaaS
•	el cliente administra su contenido sin dependencia técnica
________________________________________
11. Declaración Fundacional del Proyecto
SYNTIWeb nace para convertir la presencia digital en infraestructura automática, accesible y escalable, eliminando la dependencia técnica de los negocios y permitiendo que cualquier empresa active su web en minutos y la administre por sí misma durante toda su vida operativa.

13. Conclusión Estratégica
Separar:
•	Marca corporativa
•	Producto comercial
•	Motor tecnológico
permite que la plataforma crezca sin rediseños de marca, sin reestructuración técnica y sin fricciones comerciales.
La decisión de estructurar SYNTIWeb como producto visible y SYNTI Engine como núcleo invisible establece desde el nacimiento una base SaaS sólida, escalable y preparada para convertirse en un ecosistema completo de soluciones digitales.

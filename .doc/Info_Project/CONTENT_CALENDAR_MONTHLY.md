## 📊 MÉTRICAS A TRACKEAR

### Semanalmente:

| Métrica | Objetivo | Tracking |
|---------|----------|----------|
| **Posts publicados** | 3/semana | Manual count |
| **Stories** | 5/día | Manual count |
| **Reach promedio** | +10% s/s | IG Insights |
| **Engagement rate** | >5% | IG Insights |  
| **Link clicks** | +15/semana | Bit.ly o IG Insights |
| **DMs nuevos** | +10/semana | Manual count |
| **Demos agendadas** | 2-3/semana | Cal.com |
| **Saves** (guardados) | +20/semana | IG Insights |

### Qué significa cada métrica:

**Reach:** Cuántas personas vieron tu contenido
→ Bajo? Contenido no reaching. Prueba otro formato/hora.

**Engagement:** Likes + comments + shares / reach
→ Bajo? Contenido no resonando. Prueba otro topic.

**Link clicks:** Cuántos clickearon link en bio
→ Bajo? CTA débil o contenido no convence.

**DMs:** Interesados contactando
→ Bajo? Contenido no generando interés. Revisa value prop.

**Saves:** Guardaron tu post para después
→ Alto = Contenido valioso. Hace más de esto.
→ Bajo = Contenido no útil. Más education/tips.

##  🎯 REGLAS DE ORO CONTENIDO

### DO:

✅ **Consistencia > Perfección**
Post 70% bueno PUBLICADO > Post 100% perfecto sin publicar

✅ **Value first, sell second**
80% contenido educativo/entretenimiento
20% contenido venta directa

✅ **Engagement genuino**
Comenta con value, no "Nice post! Check my profile!"

✅ **Reuse + repurpose**
Post bueno → Reel → Story → TikTok
Una idea, 4 formatos.

✅ **Test + iterate**
Post no funciona? Try mismo mensaje, otro formato.
Post funciona? Remake con variaciones.

### DON'T:

❌ **Ghosting followers**
Comentan/DM → Dejas en visto → Pierdes lead

❌ **Spam**
Copy/paste mismo mensaje 100x → Te banean

❌ **Inconsistencia**
Post 5 veces esta semana, 0 próximas 2 semanas → Algoritmo te penaliza

❌ **Solo vender**
Todo post = "Compra compra compra" → Unfollows masivos

❌ **Comparaciones tóxicas**
"Instagram bad, web good" → Alienates gente que usa IG
Mejor: "IG + Web = Perfect combo"

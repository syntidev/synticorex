## 📊 GESTIÓN Y CRM {#gestion}

### 🗂️ Google Sheets como CRM (GRATIS) ⭐⭐⭐⭐⭐

**Por qué:**
- Cero curva aprendizaje
- Accesible desde cualquier dispositivo
- Fórmulas para automatizar
- Comparte con equipo (cuando crezcas)

**Template CRM (copiar/pegar):**

```
COLUMNAS:

A: ID (1, 2, 3...)
B: Fecha Contacto (dd/mm/yyyy)
C: Nombre Completo
D: Negocio/Marca
E: Teléfono (+58...)
F: Email
G: Instagram (@usuario)
H: Categoría (Comida, Ropa, Servicios, etc)
I: Ciudad
J: Tamaño Aprox (pequeño, mediano, grande)
K: Stage (Lead, Demo, Negociación, Cliente, Perdido)
L: Fuente (Instagram, Referido, Orgánico, Ads)
M: Plan Interesado (Oportunidad, Crecimiento, Visión)
N: Último Contacto (dd/mm/yyyy)
O: Próximo Follow-up (dd/mm/yyyy)
P: Notas
Q: Prioridad (Alta, Media, Baja)
```

**Fórmulas útiles:**

**Días desde último contacto:**
```
=TODAY()-N2
```
(Si >5 días, hacer follow-up)

**Conteo por Stage:**
```
=COUNTIF(K:K,"Cliente")
```

**Tasa conversión:**
```
=COUNTIF(K:K,"Cliente")/COUNTA(K:K)*100
```

**Colores condicionales:**

Format → Conditional formatting:
- K = "Cliente" → Verde
- K = "Demo" → Amarillo
- K = "Perdido" → Rojo
- O < TODAY() → Naranja (follow-up vencido)

**Acceso rápido móvil:**
- App: Google Sheets (Android/iOS)
- Widget en home screen para acceso 1-tap

**Costo:** $0/mes 💚

**Setup en SyntiWeb landing pages:**

1. Crear cuenta: analytics.google.com
2. Property: "SyntiWeb Client Pages"
3. Data stream: Web
4. Get **Measurement ID:** G-XXXXXXXXXX

5. **Agregar a cada tenant landing:**

```php
// resources/views/tenant/layout.blade.php

<head>
    <!-- Google tag (gtag.js) -->
    <script async src="https://www.googletagmanager.com/gtag/js?id={{ $tenant->ga_measurement_id }}"></script>
    <script>
      window.dataLayer = window.dataLayer || [];
      function gtag(){dataLayer.push(arguments);}
      gtag('js', new Date());
      gtag('config', '{{ $tenant->ga_measurement_id }}');
    </script>
</head>
```

Dashboard custom "Client Success":
- Visit count: ¿Cuánto tráfico genera el cliente?
- Bounce rate: ¿Landing atractiva? (<60% good)
- Avg session: ¿Tiempo en página? (>1min good)
- WhatsApp button clicks: ¿Conversión?

**Reportes mensuales clientes:**

```
Subject: 📊 Tu reporte mensual SyntiWeb
```

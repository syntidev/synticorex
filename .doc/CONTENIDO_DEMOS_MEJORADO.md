## 🖼️ Generación de Imágenes

Se agregó generación **automática de imágenes** para todos los productos y servicios:

### Servicios Creados:
- **`ProductImageGeneratorService`**: Genera imágenes (500x400px) para productos
  - Color basado en segmento (Azul=Tecnología, Verde=Comercio, Púrpura=Consultoría)
  - Placeholder URL con nombre del producto
  - Fallback con GD si falla descarga

- **`ServiceImageGeneratorService`**: Genera imágenes (400x300px) para servicios
  - Colores diferentes por segmento (Cyan=Tecnología, Amber=Comercio, Pink=Consultoría)
  - Idem ProductImageGeneratorService en funcionalidad

### Almacenamiento:
- **Ruta**: `storage/app/public/tenants/{tenant_id}/`
- **Formato**: JPG (85% quality)
- **Nombres**: `{slug}_{random8}.jpg` y `service_{slug}_{random8}.jpg`
- **URL de acceso**: `asset('storage/tenants/{tenant_id}/{filename}')`

## 🔄 Cómo Aplicar los Cambios

### Opción 1: Reseeding Completo (Base de datos nueva)
```bash
php artisan migrate:fresh --seed
```

### Opción 2: Actualizar datos existentes (Sin borrar BD)
```bash
# Eliminar tenants existentes
php artisan tinker
> Tenant::truncate();
> Product::truncate();
> Service::truncate();

# Ejecutar seeder
php artisan db:seed --class=TestingSeeder
```

### Opción 3: Crear nuevos tenants sin afectar existentes
```bash
php artisan tinker
> (new \Database\Seeders\TestingSeeder)->run();
```

> **Nota**: Las imágenes se generan automáticamente durante la ejecución del seeder
